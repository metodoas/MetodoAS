# Método ♠️ AS - Landing Page

## Versión 0.2 (Estado Actual - "Premium Dark")
Esta versión incorpora una estética de alta gama con efectos visuales avanzados y una estructura de ventas AIDA optimizada.

### Características Principales:
- **Header Premium Sticky:** Barra de navegación fija con efecto glassmorphism (desenfoque), borde superior rojo neón y botón CTA.
- **Efectos Neón:** Animaciones de brillo rojo en botones y elementos clave.
- **Identidad Visual:**
  - Logo "MÉTODO AS" con As de Picas (♠️) oscuro que se ilumina con un "escaneo" rojo.
  - Efecto de cursor parpadeante (`_`) en el título Hero.
  - Diagrama del método triangular con efectos interactivos de luz.
- **Copywriting AIDA:**
  - Sección de "Deseo" añadida: beneficios emocionales y transformación.
  - Botones con textos de venta directa ("ACCEDER", "QUIERO ORGANIZARME").
- **Backup:** Guardado como `index_v0.2.html`.

## Descripción General
Landing page minimalista y de alta conversión para el eBook "Método AS".

## Stack Tecnológico
- HTML5 Semántico
- CSS3 Moderno (Variables, Flexbox, Grid, Animaciones 3D)
- Sin dependencias externas (Vanilla puro)

## Estructura
- `/assets`: Imágenes y recursos.
- `index.html`: Código fuente principal.

## Despliegue
Este proyecto está optimizado para desplegarse en Netlify o Vercel simplemente arrastrando la carpeta.
